**What I Learned:** Freqtrade Hyperopt uses Bayesian search with optuna's NSGAIIISampler to optimize strategy parameters. Key insights:
- Optimizes buy/sell/stoploss/ROI/trailing parameters via many backtests
- Built-in loss functions: SharpeHyperOptLoss, SortinoHyperOptLossDaily, MaxDrawDownHyperOptLoss, ProfitDrawDownHyperOptLoss, OnlyProfitHyperOptLoss, ShortTradeDurHyperOptLoss
- Commands: `freqtrade hyperopt --strategy BBMomentum --hyperopt-loss SortinoHyperOptLossDaily --timerange 20230101-20260318 --epochs 100 --spaces buy,stoploss,roi,sell -j -1`
- Requires 2+ CPU cores, not suitable for Raspberry Pi
- Use 6+ months timerange to avoid overfitting
- Set --random-state for reproducibility
- Can create custom loss functions via IHyperOptLoss class
- Custom parameter spaces via nested HyperOpt class
- Advanced: override generate_estimator() for custom optuna samplers
