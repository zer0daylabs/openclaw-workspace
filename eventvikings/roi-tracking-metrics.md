# EventVikings Sponsorship ROI Tracking Metrics

## Overview

This document outlines the comprehensive ROI tracking system for EventVikings sponsorships. The metrics are designed to provide sponsors with clear, actionable data on their investment returns.

---

## Core Metrics

### 1. **Impressions**

**Definition**: Total number of times sponsor branding was seen

**Sources**:
- Website impressions (sponsor logo page views)
- Event page impressions
- Email impressions (sponsor features)
- Social media impressions
- Digital display impressions
- Print material impressions

**Calculation**:
```
Total Impressions = 
  (Website Views × Sponsor Logo Visibility %) +
  (Email Opens × Sponsor Feature %) +
  (Social Reach × Sponsor Mentions) +
  (Display Ad Impressions) +
  (Print Distribution × Logo Visibility)
```

**Reporting Format**:
- Total Impressions: [Number]
- Unique Impressions: [Number]
- Impressions by Source: [Breakdown]
- Impressions Timeline: [Chart]

---

### 2. **Engagement Rate**

**Definition**: Percentage of people who interacted with sponsor content

**Sources**:
- Click-through rates (CTRs)
- Social media engagement (likes, shares, comments)
- QR code scans
- Booth visits
- Email clicks
- Form submissions
- App interactions

**Calculation**:
```
Engagement Rate = (Total Engagements / Total Impressions) × 100

Total Engagements =
  Link Clicks +
  Social Interactions +
  QR Scans +
  Booth Visits +
  Email Clicks +
  Form Submissions +
  App Interactions
```

**Benchmark Targets**:
- Website CTR: 1-3%
- Email CTR: 2-5%
- Social Engagement: 3-6%
- QR Scan Rate: 5-15%
- Booth Visits: 10-20% of foot traffic

**Reporting Format**:
- Engagement Rate: [Percentage]
- Engagement by Channel: [Breakdown]
- Top Performing Assets: [List]
- Engagement Timeline: [Chart]

---

### 3. **Leads Generated**

**Definition**: Number of qualified contact information collected

**Sources**:
- QR code scans leading to lead forms
- Booth registrations
- Contest entries
- Download requests
- Demo sign-ups
- Consultation bookings

**Calculation**:
```
Total Leads = 
  QR Code Lead Forms +
  Booth Registrations +
  Contest Entries +
  Download Requests +
  Demo Sign-ups +
  Consultation Bookings
```

**Lead Quality Scoring**:
- Hot Leads (ready to buy): 100 points
- Warm Leads (interested): 75 points
- Cold Leads (potential): 50 points

**Quality Score**:
```
Quality Score = (Hot × 100 + Warm × 75 + Cold × 50) / Total Leads
```

**Reporting Format**:
- Total Leads: [Number]
- Quality Score: [Percentage]
- Leads by Source: [Breakdown]
- Lead Status: [New/Contacted/Converted]

---

### 4. **QR Code Scans**

**Definition**: Number of times sponsor QR codes were scanned

**Sources**:
- Physical QR codes on badges, materials
- Digital QR codes on event app
- Printed QR codes on promotional materials

**Calculation**:
```
Scan Rate = (Total Scans / Total Distribution) × 100

Conversion Rate = (Scans to Landing Page / Total Scans) × 100
```

**Scanning Analytics**:
- Scan count by location
- Scan timestamps (peak times)
- Device types scanning
- Referral sources

**Reporting Format**:
- Total QR Scans: [Number]
- Scan Rate: [Percentage]
- Conversion Rate: [Percentage]
- Top Scanning Locations: [Map/List]
- Scan Timeline: [Chart]

### 7. **Conversion Rate**

**Definition**: Percentage of engagements that resulted in desired action

**Sources**:
- Ticket purchases with sponsor promo codes
- Landing page conversions
- Demo bookings
- Consultation requests
- Download completions

**Calculation**:
```
Conversion Rate = (Conversions / Total Engagements) × 100
```

```
Promo Code Conversions = Redemptions with Sponsor Codes

Code: SPONSOR2026
Redemptions: 50
Revenue: $2,500
```

**Reporting Format**:

### 8. **ROI Calculation**

**Definition**: Return on Investment calculated for sponsorship

**Formula**:
```
ROI = ((Value Generated - Investment) / Investment) × 100

Value Generated = 
  (Revenue from Promo Codes) +
  (Lead Value × Quality Score) +
  (Media Value of Impressions) +
  (Brand Equity Value) +
  (Partnership Opportunities)

Investment = Sponsorship Fee + Additional Costs
```

**ROI Scenarios**:

**Conservative**: 2x ROI
- Conservative revenue estimates
- Lower lead values
- Reduced media value

**Moderate**: 2.5-3x ROI
- Realistic revenue estimates
- Standard lead values
- Average media value

**Optimistic**: 3-4x ROI
- Strong performance
- High lead quality
- Enhanced media value

**Reporting Format**:
- Total Investment: $[Amount]
- Total Value Generated: $[Amount]
- Net Return: $[Amount]
- ROI: [Percentage]
- Break-even Point: [Timeline]
- Comparison to Industry Average: [Higher/Lower]

## Additional Metrics

### **Cost Per Lead (CPL)**
```
CPL = Total Investment / Total Leads

Example: $15,000 / 150 leads = $100 per lead
```

### **Cost Per Impression (CPM)**
```
CPM = (Total Investment / Total Impressions) × 1,000

Example: $15,000 / 25,000 impressions = $0.60 CPM
```

### **Cost Per Engagement (CPE)**
```
CPE = Total Investment / Total Engagements

Example: $15,000 / 3,500 engagements = $4.29 per engagement
```

### **Share of Voice**
```
Share of Voice = (Your Brand Mentions / Total Brand Mentions) × 100

Example: 45 mentions / 150 total mentions = 30% share of voice
```

### **Brand Lift**
- Pre-event awareness survey: [Percentage]
- Post-event awareness survey: [Percentage]
- Lift: [Percentage point increase]

## Data Collection Methods

### **Automated Tracking**:
1. **QR Code Scanning**: Unique codes per sponsor, real-time tracking
2. **Promo Code Tracking**: Custom codes for each sponsor
3. **Website Analytics**: Pixel tracking on sponsor pages
4. **Email Tracking**: Open and click tracking
5. **Social Listening**: Automated mention tracking
6. **Booth Scanners**: QR code scanning at sponsor booths

### **Manual Data Collection**:
1. **Surveys**: Post-event attendee surveys
2. **Interviews**: Sponsor follow-up calls
3. **Feedback Forms**: Event feedback collection
4. **Focus Groups**: Optional deep-dive sessions

### **Third-Party Integrations**:
1. **CRM Integration**: Salesforce, HubSpot, etc.
2. **Analytics Tools**: Google Analytics, Adobe Analytics
3. **Social Platforms**: Native analytics from each platform
4. **Payment Processors**: Stripe, PayPal transaction data
