## 🎮 How to Use the System

### Log Events (Real-time)
```bash
# After learning something
~/.openclaw/workspace/bin/graphiti-log.sh "Lesson: Query knowledge before implementing"

# After making a decision
~/.openclaw/workspace/bin/graphiti-log.sh "Decision: Use local vault for API keys - security first approach"

# After achieving milestone
~/.openclaw/workspace/bin/graphiti-log.sh "Milestone: Graphiti system operational with 20+ facts stored"
```

### Query Knowledge (Before Decisions)
```bash
# Review past lessons
~/.openclaw/workspace/bin/graphiti-query.sh "lessons about API key management"

# Check past decisions
~/.openclaw/workspace/bin/graphiti-query.sh "decisions on password management"

# See available capabilities
~/.openclaw/workspace/bin/graphiti-query.sh "agent skills capabilities"

# Query project status
~/.openclaw/workspace/bin/graphiti-query.sh "MusicGen repository status"
```

### Query-Before-Acting Pattern
```bash
# 1. Query past patterns
~/.openclaw/workspace/bin/graphiti-query.sh "lessons about {topic}"

# 2. Review and apply insights
# 3. Make informed decision
# 4. Log the decision
~/.openclaw/workspace/bin/graphiti-log.sh "Decision: {rationale}"
```
