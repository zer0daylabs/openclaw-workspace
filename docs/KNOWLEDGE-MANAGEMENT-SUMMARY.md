## How to Use

### For Team Members
1. **Quick Start**: Read `docs/OPENCLAW-INTEGRATION.md`
2. **Understand System**: Review `memory/openclaw-research.md`
3. **Reference Documentation**: Use Slack Canvas structure for quick access
4. **Learn Best Practices**: Study `memory/self-improving-agents-overview.md`

### For System Administration
1. **Set Up Workspace**: Follow integration guide steps 1-6
2. **Configure Skills**: Review `skills/*/SKILL.md` for each capability
3. **Establish Memory**: Set up daily log and weekly curation
4. **Enable Heartbeat**: Configure `HEARTBEAT.md` for proactive monitoring
5. **Setup Security**: Configure passwords vault, Bitwarden as needed

### For Knowledge Management
1. **Run Batch Script**: When Graphiti operational, execute `bash scripts/batch-add-knowledge.sh`
2. **Verify Nodes**: Use query patterns to confirm import
3. **Establish Sync**: Schedule weekly knowledge updates
4. **Monitor Evolution**: Track agent learning and improvements

### For Demo/Training
1. **Execute Demo**: Run `bash scripts/demo-fetch-github-issues.sh`
2. **Review Output**: Check `memory/demo-issues-output.md`
3. **Verify Results**: Use verification steps provided
4. **Discuss Patterns**: Review captured issues and patterns
