## Lessons Learned & Best Practices

### 1. **File-Based Memory System**
**Pattern:** Daily logs + long-term memory  
**Value:** Creates continuity across sessions  
**Implementation:**
- `memory/YYYY-MM-DD.md` — raw session logs
- `MEMORY.md` — curated wisdom
- `AGENTS.md` — operating procedures
- `TOOLS.md` — infrastructure notes

### 2. **Agent Orchestration Patterns**
**Pattern:** Decompose → Spawn → Monitor → Consolidate  
**Value:** Enables complex multi-agent workflows  
**Components:**
- Sub-agent workspaces with inbox/outbox
- Status tracking via JSON
- Dynamic SKILL.md generation
- Clean agent dissolution

### 3. **Trust & Transparency**
**Principle:** Never lie or hide anything  
**Practice:** Direct communication, no surprises  
**Benefit:** Builds long-term trust and reliability

### 4. **Resourceful Problem-Solving**
**Pattern:** Read, check context, search → then ask  
**Value:** Reduces dependency, increases autonomy  
**Result:** Comes back with answers, not questions

### 5. **Boundary Management**
**Rule:** Bold internally, careful externally  
**Application:**
- Safe: Reading, organizing, learning, internal actions
- Ask first: Emails, tweets, public posts, anything leaving machine

### 6. **AI Cost Efficiency**
**Achievement:** ~$0.05 total AI costs  
**Method:** Caching, efficient tool use, avoiding redundant calls
